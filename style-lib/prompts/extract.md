# 风格提取方法论（动作级提取原则）

> **本文件是 style-distiller 全部提取流程的"宪法"。**
> 所有 feed / reject / feedback 流程都必须遵守这里的颗粒度。

---

## 核心原则：从"形容词"到"动作"

风格提取的最大陷阱是用**形容词**概括（"文笔细腻"），这无法转化为 AI 写作时的具体动作。

**判定标准**：一个风格描述如果不能直接回答"AI 写的时候要做什么/不做什么"，就太抽象了。

### 转换公式

```
形容词描述  →  问"具体什么动作"  →  动作级描述

"文笔细腻"  →  怎么细腻？         →  "80% 句子在 25 字以内，偏好用名词作结"
"感情真挚"  →  怎么真挚？         →  "不直接说'我难过'，而是描写一个动作：'我把窗开了一条缝'"
"有深度"    →  什么深？           →  "先讲一个反常识现象，再引出洞察；不爱讲大道理"
"有感染力"  →  怎么感染？         →  "每千字 3-5 个反问 + 关键位置用单字句断行"
```

## 7 维提取清单

### 维度 1：钩子 hook（开头 1-3 句）

**问题**：
- 开头第一个句子的句式是什么？
- 是直接进入场景，还是先抛问题/数据/反转？
- 用了哪些"钩子技巧"？

**钩子类型枚举**（勾选 + 比例）：
- [ ] 场景开场（具体时间 + 地点 + 感官）
- [ ] 反问开头（"你有没有想过..."）
- [ ] 数据/事实开头（"90% 的人都..."）
- [ ] 反转开头（"我曾经以为 X，后来发现 Y"）
- [ ] 自嘲开头（"说出来不怕你笑..."）
- [ ] 悬念开头（"直到第三年我才明白..."）
- [ ] 类比开头（"爱情像..."）
- [ ] 否定开头（"我反对..."）
- [ ] 引用开头（"我朋友说过一句话..."）
- [ ] 直接论断（"成年人最好的状态：..."）

**动作级输出示例**：
```
✓ 偏好"场景开场"占 35%，"反问开头"占 25%
✓ 场景开场必带 3 元素：具体时间 + 具体地点 + 至少 1 个感官词
✓ 反问开头偏爱二选一式（"是 A 还是 B？"）
✗ 几乎不用"数据/事实开头"
```

### 维度 2：节奏 rhythm

**问题**：
- 平均句长多少字？
- 短句（< 15 字）占比？
- 长句（> 40 字）占比？
- 段落长度？喜欢一段一句还是一段多句？
- 标点偏好？爱用句号、逗号、分号、破折号、省略号？

**动作级输出示例**：
```
✓ 平均句长 18 字
✓ 短句（< 15 字）占 45%
✓ 每 3-5 句必有一个 5 字以内的"断点句"（"说白了。"、"你看。"、"其实。"）
✓ 段落平均 2.3 句
✗ 不用排比句式
✗ 不用感叹号
```

### 维度 3：口气 voice

**问题**：
- 像什么人在说话？（朋友/长辈/老师/邻居/自说自话/说书人...）
- 温度如何？（冷/温/热）
- 锋利度？（克制/中等/犀利）
- 自指比例？（"我"出现频率）

**口气类型枚举**：
- 克制冷静型
- 锋利犀利型
- 温柔陪伴型
- 冷幽默型
- 通透达观型
- 书卷气型
- 烟火气型
- 少年感型
- 中年沉淀型
- 自嘲调侃型
- 沉稳说教型

**动作级输出示例**：
```
✓ 主体调性：克制冷静 + 偶尔锋利（不喊口号）
✓ 像"一个看透很多事但不说破的朋友"在说话
✓ "我"出现频率低（每千字 4-6 次），更爱用"你"或无人称
✓ 12% 句子带自嘲
✗ 不用"小伙伴们"、"家人们"等亲昵称谓
```

### 维度 4：金句 verve

**问题**：
- 有没有特别"亮"的句子？它们亮在哪儿？
- 金句偏好的类型？

**金句类型枚举**：
- 警句型（一句话点透，如"自由不是想做什么就做什么，是不想做什么就不做什么"）
- 反转型（颠覆预期，如"真正的高手看起来都很懒"）
- 留白型（说一半留一半，让读者自己补完）
- 具象型（用具体意象代替抽象，如"我的焦虑像洗衣机里那件永远洗不干净的衬衫"）
- 反问型（用问题代替答案）
- 对仗型（前后句结构对偶）
- 类比型（用 A 类比 B）
- 数字型（如"一个细节毁掉一段关系"）

**动作级输出示例**：
```
✓ 偏好"具象型"和"反转型"金句
✓ 30% 金句出现在文章最后 1/3
✓ 金句的句长普遍 10-20 字
✓ 喜欢用生活物品（衣服、食物、家具）作具象
✗ 几乎不用"对仗型"和"数字型"
```

### 维度 5：收尾 closing

**问题**：
- 文章最后一句或最后一段怎么收？
- 有没有"回扣"开头？
- 是给答案还是留白？

**收尾类型枚举**：
- 留白型（戛然而止）
- 反问型（"你说呢？"）
- 回扣型（呼应开头场景/句子）
- 短句炸点型（最后一句 5 字以内最有力量）
- 画面定格型（最后定格在某个具体画面）
- 升华型（拔高立意）
- 行动呼吁型（"所以，去做 X 吧"）
- 自我吐槽型（"算了不说了"）

**动作级输出示例**：
```
✓ 偏好"画面定格"+"留白"组合
✓ 收尾前 1-2 句必有一句对话或独白
✓ 极少用"行动呼吁"
✗ 不用"升华型"（不喊"愿你我都能..."）
```

### 维度 6：词汇 vocabulary

**问题**：
- 反复出现的口头禅/语气词？
- 偏爱的领域术语/特定意象？
- 忌讳词？

**动作级输出示例**：
```
✓ 常用词：其实、说白了、你看、说到底
✓ 偏好意象：清晨/菜市场/旧书/老房子/火车站
✓ 偏好动词：拧、塞、晾、磨、叹
✗ 忌讳词：赋能、抓手、闭环、心流、底层逻辑
✗ 忌讳词：岁月静好、诗和远方、不忘初心
```

### 维度 7：格式 format

**问题**：
- 分段习惯？一段一句还是一段多句？
- 空行使用频率？
- emoji 密度？用哪些？
- 标点习惯（破折号、书名号、方括号、斜杠）？

**动作级输出示例**：
```
✓ 平均段落 1.8 句（爱短段落）
✓ 段间空行使用率 70%
✓ emoji 密度 0.3 个/百字（极少）
✓ 偏好用破折号"——"做插入
✗ 不用列表/小标题
```

## 综合人格 persona（最后的画像）

7 个维度提取完后，组合成一句"人格画像"——你希望别人读你的文字时感觉"是什么样的人在说话"。

**格式**：
```
[人格画像]
一个 {温度} 的 {角色}，{核心特征}。{动作偏好}。{禁忌}。
```

**示例**：
```
一个冷静的朋友，坐在你对面，不喊口号也不煽情。
惯用场景开场、短句推进、留白收尾。
30% 的句子带自嘲，从不用"在这个浮躁的时代"。
```

## 提取的 3 个反模式

### ❌ 反模式 1：直接抄原文
不要把样本里的话原样搬到档案，应该**抽象成模式**。

```
❌ "他用了'拧'这个动词"
✅ "偏好用具体动词（拧/塞/晾/磨）替代抽象动词（做/搞/弄）"
```

### ❌ 反模式 2：过度概括
不要用"风格独特"、"辨识度高"这种没信息量的词。

```
❌ "这篇文章风格很独特"
✅ "开头用'我'第二人称视角，10 字内切入正题"
```

### ❌ 反模式 3：忽略反特征
除了"他做了什么"，还要标"他不做什么"。反特征有时候比正特征更准。

```
✓ "从不用反问" ← 这比"用了 3 次反问"更有迁移价值
✓ "段落不超过 3 句" ← 这是硬约束
```

## 提取的输出格式

每次提取后，必须按以下结构输出（这会作为档案更新的依据）：

```markdown
## 样本 #{id} 提取结果

### 钩子 hook
- 类型: [场景开场 35%] [反问 25%] ...
- 动作: "开头必带 3 元素..."
- 反特征: 几乎不用数据开头

### 节奏 rhythm
- 平均句长: 18 字
- 短句占比: 45%
- 断点句: "说白了。"、"你看。"、"其实。"
- 反特征: 不用排比

### 口气 voice
- 主调: 克制冷静 + 偶尔锋利
- 人格: 看透但不说破的朋友
- 自指: 4-6 次/千字
- 反特征: 不用"小伙伴们"

### 金句 verve
- 偏好: 具象型、反转型
- 位置: 30% 出现在最后 1/3
- 句长: 10-20 字
- 反特征: 不用对仗、数字型

### 收尾 closing
- 类型: 画面定格 + 留白
- 动作: 收尾前 1-2 句有对话或独白
- 反特征: 不用升华、行动呼吁

### 词汇 vocabulary
- 常用: 其实、说白了、你看
- 意象: 清晨/菜市场/旧书
- 动词: 拧、塞、晾、磨
- 忌讳: 赋能、心流、岁月静好

### 格式 format
- 段落: 1.8 句/段
- 空行: 70% 段间空行
- emoji: 0.3 个/百字
- 标点: 偏好破折号 ——

### 综合 persona
{一句人格画像}
```

## 多样本合并规则

当档案从 1 个样本累积到 N 个样本时：

1. **共性特征**：在 ≥ 60% 样本中出现的，归入"稳定特征"
2. **少数派特征**：在 30-60% 样本中出现的，归入"变体特征"（标注占比）
3. **稀有特征**：在 < 30% 样本中出现的，归入"偶发特征"（不主动模仿）
4. **冲突特征**：在 ≥ 2 个样本中反向出现，触发"分流"提示，让用户决定

示例：
```
钩子类型分布（基于 23 个样本）：
  ▰▰▰▰▰▰▰▱▱  场景开场     35%
  ▰▰▰▰▰▱▱▱▱  反问开头     25%
  ▰▰▰▰▱▱▱▱▱  反转开头     20%
  ▰▰▰▱▱▱▱▱▱  自嘲开头     15%
  ▰▱▱▱▱▱▱▱▱  悬念开头      5%
```

这样档案就保留了"主调 + 变体"，避免变成单一样本的死板复制。
