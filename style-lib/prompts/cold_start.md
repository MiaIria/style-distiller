# 冷启动引导（🔴 状态）

> 当 `weights.json` 显示 `current_state: cold_start`（0-2 个正样本）时执行。

## 引导原则

- **不要硬塞样本**：让用户在 3 分钟内走完引导
- **采 3 颗种子**：3 个问题替代"先喂 5 篇"
- **当场出第一篇**：引导结束后立刻生成 1 篇，让用户看到效果
- **不完美也没关系**：标注"风格初形成"即可

## 流程

### Step 1：欢迎与状态说明

```
🌱 你的风格档案刚开始积累。

目前 0 样本，状态：🔴 冷启动。

要解锁"像你写"的能力，最少需要 3 颗种子样本。
我设计了一个 3 分钟的引导：3 个问题，1 篇试写。

准备好了吗？
```

### Step 2：3 个引导问题

用 `AskUserQuestion` 一次性问 3 个问题（多选 + 单选混合）：

**问题 1：你的身份画像**
```
你日常会写什么类型的文字？（多选）

  [ ] 朋友圈 / 微博（私人表达）
  [ ] 小红书 / 即刻（生活分享）
  [ ] 公众号 / 博客（深度内容）
  [ ] 工作文档 / 邮件（职业写作）
  [ ] 都不太写，主要看别人的
```

**问题 2：你的语气偏好**
```
你希望你的文字给人什么感觉？（单选）

  [ ] 克制冷静（看透不说破）
  [ ] 锋利犀利（一针见血）
  [ ] 温柔陪伴（共情细腻）
  [ ] 冷幽默（举重若轻）
  [ ] 通透达观（看开了）
  [ ] 烟火气（贴近生活）
  [ ] 书卷气（有学识感）
  [ ] 自嘲调侃（先笑自己）
```

**问题 3：你的"反例"**
```
你特别反感哪种调调？（单选，避免日后写出这种东西）

  [ ] 鸡汤煽情（"愿你..."）
  [ ] 油腻说教（"年轻人应该..."）
  [ ] 套路化抒情（"在这个浮躁的时代"）
  [ ] 强行升华（最后一句拔高立意）
  [ ] 空洞排比（"我们要... 我们要... 我们要..."）
  [ ] 网络梗堆砌（"yyds"、"绝绝子"）
```

### Step 3：填充基础档案

基于 3 个问题，写入初始 profile：

#### profile/voice.md
- 在"主体调性"勾选用户选的 1-2 个
- 温度/锋利度按选项设默认位置

#### profile/vocabulary.md
- 忌讳词区域按"反例"选项预填：
  - "鸡汤煽情" → 禁用"愿你"、"岁月静好"
  - "油腻说教" → 禁用"年轻人应该"、"我们要"
  - 等等

#### profile/closing.md
- 收尾类型按"反例"打 ✗：
  - 选"强行升华" → 升华型标 ✗
  - 等等

#### profile/persona.md
- 一句话画像先填"通用基线"：
  > 一个偏好 {主调} 的写作者，惯用 {动作}，避免 {反例}。
- 状态保持 🔴 冷启动
- 标注"基线画像，等样本丰富后自动重写"

### Step 4：完成引导对话

```
🌱 基础档案已生成！

📝 临时画像：一个偏好【{用户选的主调}】的写作者，
              惯用【{推断的动作}】，避免【{反例}】。

💡 接下来有 2 条路：

  [1] 直接 /style-write 试试
      现在还是"通用基线 + 你的偏好偏移"，但能看到雏形

  [2] 喂 1-2 篇你喜欢的文章（/style-feed）
      解锁真正的"像你写"能力

  [3] 喂 1 篇你自己写的文章（标 source: self）
      这是最强信号，1 篇自我样本 = 3 篇外部样本

推荐：先 1 篇自我样本 + 1 篇外部样本，5 分钟搞定，状态就变 🟠 萌芽了。
```

### Step 5：等待用户行动

- 用户选 [1] → 转 /style-write（标注"基线模式"）
- 用户选 [2] / [3] → 转 /style-feed

---

## 边界情况

### 1. 用户回答含糊
- 用 free-text 追问，不阻塞流程

### 2. 用户嫌问题太多
- 缩减到 1 个问题（只问"语气偏好"），其他用默认值
- 默认反例 = "鸡汤煽情"（最常见反感）

### 3. 用户跳过引导直接 /style-write
- 仍然按"通用基线"生成
- 显著标注"⚠️ 档案为空，使用通用风格"

---

## 状态提升的最低门槛

- 🔴 冷启动 → 🟠 萌芽：1 个正样本
- 🟠 萌芽 → 🟡 学习：10 个正样本
- 🟡 学习 → 🟢 成熟：30 个正样本

自我样本（source: self）权重 = 1.2，喂 1 篇自我 + 1 篇外部就能解锁 🟠。
